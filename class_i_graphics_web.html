“background.png”) and type (e.g. “PNG”) On macOS resources are usually included inside the bundle resources folder. In that case you provide a filename and this method will return the absolute path to the resource. In some cases you may want to provide an absolute path to a file in a shared resources folder here (for example if you want to reduce the disk footprint of multiple bundles, such as when you have multiple plug-in formats installed). On Windows resources are usually baked into the binary via the resource compiler. In this case the filename argument is the resource id.
